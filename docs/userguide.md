# User Guide

## 🚀 How to Run the Application

1. Clone the repository:
   git clone https://github.com/Devanshu-MUJ/devopsprojectpullrequestdashboard.git

2. Navigate to the project folder:
   cd devopsprojectpullrequestdashboard

3. Run using Docker:
   docker-compose up --build

4. Open browser:
   http://localhost:8080

---

## 🧑‍💻 How to Use

1. Enter GitHub Username
2. Enter Repository Name
3. Click "Fetch PRs"
4. View pull request details

---

## 📊 Output Information

* Pull Request Number
* Title
* Author
* Status
* Total PR count

---

## ⚠️ Notes

* Works only for public repositories
* Requires internet connection
* API may limit excessive requests

---

## 🛠️ Troubleshooting

* If Docker fails, ensure Docker Desktop is running
* If no data appears, check repository name
* Restart container if needed

---

## 📌 Conclusion

This tool helps users easily monitor GitHub pull requests in a simple and efficient way.
